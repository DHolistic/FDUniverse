================================================================================
                      DEVELOPMENT WORKFLOW
================================================================================

To make changes:
  1. Edit HTML/JS/CSS directly in 08_interactive_reader/
  2. Test in browser: Open HTML file directly or use dev server
  3. Git commit when ready: git add . && git commit -m "description"

To add content:
  1. Add manuscript chapters to _manuscripts/ folder
  2. Update chapter-data.json with chapter references
  3. Add images to _graphic_novel/06_generated_panels/
  4. Test in horizontal-reader or basic reader

To modify styling:
  1. Update CSS custom properties in :root section
  2. Modify component-specific CSS as needed
  3. Test responsiveness at multiple viewport sizes

Current Status:
  Branch: main
  Recent work: Chapter refactoring, nav edits, grid optimization
  Next focus: View/edit interface, animation enhancements
